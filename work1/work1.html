Сброс стилей в css (CSS Reset) – это, по сути, обнуление основных свойств и атрибутов, установленных браузером по умолчанию для различных элементов страницы, например, окно браузера может иметь ненужные вам отступы и т.п. Чтобы обнулить все заданные свойства по умолчанию, т.е. выполнить сброс стилей, мы должны начать формировать наши таблицы стилей с прописывания определенного набора стилей.
